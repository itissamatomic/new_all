# Personal Portfolio Site

A minimal Jekyll-based personal portfolio and bio site.

## Setup

1. Install Ruby (3.0+) and Bundler
2. Run `bundle install`
3. Run `bundle exec jekyll serve`
4. Visit `http://localhost:4000`

## Customization

- Edit `_config.yml` to update site title and metadata
- Update `index.md` for homepage content
- Edit `about.md` for your bio
- Add new posts in `_posts/` directory

## Deploy

Push to GitHub and enable GitHub Pages in repository settings to auto-deploy.
